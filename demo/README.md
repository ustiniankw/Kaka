# Kaka · Web 试玩版（iOS-styled）

一个纯前端（HTML + Canvas）的 Kaka 桌面宠物行为模拟器，采用 iOS 系统设计语言。

## 🎨 UI 风格

- **iOS 系统色**：`#F2F2F7` 背景 / SF 家族字体 / 系统蓝 `#007AFF`
- **分组列表**：iOS Settings 里的 `list-group + row` 样式
- **iOS 开关**：重力、显示 Kaka 使用系统 switch
- **系统胶囊 Toast**：所有反馈居中弹胶囊
- **深色模式自适应**：跟随系统 `prefers-color-scheme`

## 🎭 内含性格系统

| 性格 | 特点速览 |
|---|---|
| 🫥 社恐 | 走得慢、大部分时间发呆 |
| 🗣️ 话痨 | 时不时冒对话气泡 |
| 😴 懒鬼 | 走路慢、饥饿慢、拉得少 |
| 🍗 干饭王 | 冲食物很快、拉得多、吃得多 |
| 😤 暴躁 | 心情掉得快、掉底会瞪眼 |
| 😳 傲娇 | 摸头 30% 概率拒绝 |
| 🥳 社牛 | 满屏跑，撞墙会挥手 |

每种性格调节：走路速度、走 vs 空闲概率、拉屎频率、饥饿/心情衰减、特殊互动。

## ✨ 支持的手感

- 🚶 随机行走 / 空闲
- 🌍 重力开关（iOS switch）
- 💩 定时拉屎，`💩 立刻拉` 手动触发
- 🧹 单击 💩/💦 清理
- 🍪 喂食：丢食物 → Kaka 走过去吃
- 👋 拖拽 Kaka 扔飞
- 🕶 一键隐藏
- 📊 实时状态条：饥饿 / 心情 / 卫生 / 好感度

## 部署

```bash
cd demo
python3 -m http.server 8000
# open http://localhost:8000
```

或直接双击 `index.html`。
