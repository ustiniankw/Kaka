📦 安装 Chrome 扩展

当前为开发者版本，需通过“加载已解压的扩展程序”的方式安装：

  1. 在 GitHub 上打开仓库：github.com/ustiniankw/Kaka
  2. 点击 Code › Download ZIP 下载源码并解压
  3. 打开浏览器，访问 chrome://extensions/
  4. 右上角开启 开发者模式
  5. 点击 “加载已解压的扩展程序”
  6. 选择解压后的仓库中的 extension/ 目录

成功后：右上角会出现 Kaka — 桌面宠物 图标，任意页面右下角会注入一个小 Kaka。
